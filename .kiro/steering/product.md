# Sprint Room — product steering

Sprint Room is a **multiplayer planning workspace**: humans + one visible AI teammate turn messy ideas into a sprint-ready artifact in one session.

## Non-negotiables

- Shared room context is the source of truth (Yjs). Never treat AI as a private 1:1 chat.
- Join with a display name only — no accounts for MVP.
- Cap: 3 humans + Sprint AI.
- AI actions: Clarify → Plan → Break down. Outputs land in the shared doc for everyone to edit.
- Leave with an exportable artifact (Markdown / JSON / PRD / GitHub Issues / Checklist) from the sprint packet.

## Demo happy path (≤90s)

1. Create room → copy link → second browser joins
2. Both add 2–3 raw inputs
3. Clarify → answer 1–2 questions
4. Plan → skim/edit packet
5. Export Markdown (or PRD / Issues)

## Explicit non-goals

Full PM tooling, auth/SSO, multi-agent rooms, mobile-native apps.
